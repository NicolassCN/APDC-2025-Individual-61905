APDC Individual Project

Welcome to the APDC Individual Project API. This is a RESTful web service that provides user management functionality with role-based access control.

Available Endpoints

Base URL: /rest

Authentication

  • POST /user/register
    Register a new user
  • POST /user/login
    Login with username/email and password
  • POST /user/logout
    Logout current user

User Management

  • GET /user/{username}
    Get user information
  • PUT /user/{username}/role
    Change user role (ADMIN only)
  • PUT /user/{username}/status
    Toggle user status (GBO and above)
  • GET /user/list
    List all users (GBO and above)

Role Hierarchy

The system implements the following role hierarchy (from highest to lowest privileges):

  • ADMIN - Full system access
  • SU (Super User) - Advanced system management
  • GS (Global System) - System-wide operations
  • GBO (Global Back Office) - Back office operations
  • USER - Basic user access
